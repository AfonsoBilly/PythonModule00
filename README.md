# Growing Code — Python Fundamentals Through Garden Data

**Growing Code** is a 42 School Python fundamentals project focused on learning core programming concepts through practical, real-world exercises. From basic input/output to recursion and type annotations, each exercise builds on the last while keeping the code clean, readable, and flake8 compliant.

---

## Description

This project introduces Python's fundamental building blocks through community garden scenarios. Each exercise covers a key concept:

- **ex0** — `ft_hello_garden`: printing a message
- **ex1** — `ft_garden_name`: user input and string output
- **ex2** — `ft_plot_area`: arithmetic and variables
- **ex3** — `ft_harvest_total`: accumulating values
- **ex4** — `ft_plant_age`: conditionals
- **ex5** — `ft_water_reminder`: conditionals with thresholds
- **ex6** — `ft_count_harvest`: loops (iterative and recursive)
- **ex7** — `ft_seed_inventory`: type annotations, string methods, and unit handling

---

## Instructions

### Requirements

- Python 3.10+
- flake8 (linter)
- mypy (type checker, required for ex7)

### Installation

```bash
pip install flake8 mypy
```

### Running

Each exercise is a standalone Python file. To test your functions, use the provided helper:

```bash
python3 main.py
```

To lint your code:

```bash
flake8 ft_hello_garden.py
```

To check types (ex7):

```bash
mypy ft_seed_inventory.py
```
